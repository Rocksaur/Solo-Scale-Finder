# Solo Scale Finder

**by Dongyeol Lee**

A browser-based HTML app for exploring solo scales over chord progressions.
Users can build progressions bar by bar, analyze scale candidates, preview them
on guitar fretboard and piano keyboard, change tuning, transpose the whole song,
and listen to selected scales.

## Features

- Bar-by-bar chord progression builder
- Scale recommendations based on harmonic context
- Full candidate list including context-aware alternatives
- Guitar fretboard visualization
- Piano keyboard visualization
- Multiple guitar tuning presets
- Whole-song transposition controls
- Multilingual interface
- Audio playback for selected scales

## Running the app

This project is a static HTML app.

1. Download or clone the repository.
2. Open the main HTML file in a modern web browser.
3. Use the chord builder to enter a progression and click **Analyze!**

For best results, use an up-to-date desktop or mobile browser.
